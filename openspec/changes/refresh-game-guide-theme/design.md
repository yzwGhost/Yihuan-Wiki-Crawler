# Design

## Visual Direction

Use a dark “character dossier” aesthetic:

- Brass and ember accents instead of generic blue SaaS tones
- Layered background with vignette and grid texture
- Side navigation that feels like a game codex panel
- Cards and drawers styled like archive plates
- Tables and JSON panels tuned for readability but still atmospheric

## Scope

This change focuses on shared shell styling and existing pages:

- App layout
- Placeholder pages
- Crawl task page
- Character list page
- Character detail drawer
- Shared utility classes in the renderer stylesheet

No data flow or IPC behavior changes are included.
