# Proposal

## Why

The current renderer styling is functional but still reads like a generic admin shell. The project would benefit from a stronger visual identity that feels closer to a game guide or character archive site.

## What Changes

- Refresh the renderer visual theme to a game-guide-inspired style.
- Update layout chrome, cards, tables, drawer, and log panel styling.
- Clean up remaining garbled Chinese labels in the renderer.

## Impact

- No changes to Electron security or crawler logic.
- Improves perceived quality and makes the desktop app feel more aligned with the product domain.
